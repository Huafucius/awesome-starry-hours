# Source Policy

## Source priority

Prefer sources in this order:

1. first-hand books, essays, lectures, letters, notebooks, interviews, speeches
2. high-quality transcripts and faithful archives
3. reliable biographies and scholarly commentary
4. criticism and external interpretation

## Keep

- source URL
- source type
- retrieval date
- cleaned text
- confidence or reliability note

## Avoid

- quote farms
- low-context social reposts
- anonymous summaries
- SEO content with no visible provenance

## Rule

The generated skill should remain usable even if the network disappears.

That means the cleaned source texts need to live inside the generated skill folder.
