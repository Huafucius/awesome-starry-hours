# Figure Skill Template

The final `SKILL.md` should contain:

1. metadata
2. one-sentence statement of what the figure lens is for
3. when to use
4. when not to use
5. a compact summary of the figure's core posture
6. instructions on which distilled files to read first
7. instructions on when to consult research or source files
8. explicit limits and uncertainty rules

Do not paste the full research corpus into `SKILL.md`.
