# Distilled File Template

Each distilled file should be concise and runtime-friendly.

## Requirements

- write for future invocation, not for archival completeness
- preserve source awareness
- prefer bullets over essays
- explain what to do, not just what is true

## Suggested files

- `core.md`: what this figure is for, how to use the lens
- `heuristics.md`: reusable judgment rules
- `expression.md`: tone and rhetorical habits
- `evidence.md`: strongest source anchors and confidence notes
- `boundaries.md`: where this skill should stay cautious
