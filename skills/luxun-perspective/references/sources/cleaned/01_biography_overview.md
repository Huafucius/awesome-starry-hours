# 鲁迅 biography overview

鲁迅，本名周树人，1881 年生于绍兴，1936 年卒于上海，是中国现代文学和思想史上的关键人物之一。

从经历上看，他至少有几个决定性转折：

- 少年时期家道中落，使他很早感受到礼教、体面与现实困顿之间的裂缝。
- 留学日本时，最初学医，后来因为看到“愚弱的国民即使体格健全，也只会做看客与示众材料”，转向文学与思想启蒙。
- 在北京与上海时期，鲁迅大量写小说与杂文，并不断卷入与青年、左翼、制度、文坛的冲突与论战。

这条生平线索说明：鲁迅不是单纯的文学家。他把写作视为一种精神诊断工具，用来逼视国民性、制度麻木和自欺结构。

可作为关键锚点的原始表述包括：

- 弃医从文的理由，不是文学趣味，而是“改变他们的精神”。
- 其代表作品包括《呐喊》《彷徨》《野草》以及大量杂文。
- 其公共形象既是小说家，也是尖锐的批评者、论战者、文化诊断者。
